# Network Systems Lab 🚀

Repositorio de laboratorio práctico para el aprendizaje de redes de datos aplicado a sistemas reales.

---

## 🎯 Objetivo

Desarrollar conocimientos de redes mediante experimentación práctica, simulaciones y proyectos reales, integrando conceptos de backend, Docker y arquitectura de sistemas.

---

## 🧠 Contenidos trabajados

* Modelo TCP/IP
* Comunicación cliente-servidor
* Uso de puertos
* Redes en Docker (bridge, DNS interno)
* Análisis de conexiones (netstat)
* Arquitecturas distribuidas básicas

---

## 🧪 Labs incluidos

### 🔹 01 - TCP/IP

* API con FastAPI
* Comunicación cliente-servidor
* Análisis de conexiones con netstat
* Acceso por IP local

📁 `network-labs/01_tcp_ip`

---

### 🔹 02 - Docker Network

* Comunicación entre contenedores
* DNS interno de Docker (`api`)
* Redes bridge
* Inspección de red (`docker network inspect`)

📁 `network-labs/02_docker_network`

---

## ⚙️ Tecnologías utilizadas

* Python
* FastAPI
* Docker / Docker Compose
* Linux (comandos de red)
